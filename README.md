# Guedes IA

Este projeto já está pronto para subir na internet como site estático.

## O que ele faz

- Responde perguntas comuns de forma direta.
- Mostra links de consulta em fontes confiáveis.
- Tenta buscar respostas públicas antes de usar o fallback local.
- Funciona em celular e computador.

## Como publicar

### Opção 1: GitHub Pages

1. Crie um repositório no GitHub.
2. Envie este arquivo `index.html` para a raiz do repositório.
3. Vá em `Settings > Pages`.
4. Em `Build and deployment`, escolha `Deploy from a branch`.
5. Selecione a branch `main` e a pasta `/root`.
6. Salve e aguarde o link do site.

### Opção 2: Netlify

1. Acesse https://www.netlify.com
2. Faça login.
3. Arraste a pasta do projeto para a área de deploy.
4. O Netlify gera um link público na hora.

### Opção 3: Vercel

1. Acesse https://vercel.com
2. Entre com sua conta.
3. Importe o projeto ou arraste a pasta.
4. Publique o site.

## Observação importante

Para respostas com fontes confiáveis, é melhor publicar o site na internet em vez de abrir só como arquivo local, porque algumas buscas externas funcionam melhor fora do modo `file://`.
