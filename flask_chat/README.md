# Guedes IA — Protótipo Flask

Rápido protótipo de chat usando Flask. O backend usa respostas diretas definidas e um fallback sintetizado.

Como executar (Python 3.8+):

```bash
python -m venv venv
venv\Scripts\activate   # Windows
pip install -r requirements.txt
python app.py
```

Abra http://127.0.0.1:5000 no navegador.
